== 개요
Hospital API는 환자 정보를 관리하는 RESTful API입니다.

== 공통 응답 구조

=== 성공 응답
요청이 성공하면 JSON 형식으로 결과를 반환합니다.

=== 실패 응답
모든 API는 실패 시 아래와 같은 형식의 에러 응답을 반환합니다.

.실패 응답 예시
[source,json]
----
{
  "timestamp": "2025-08-23T12:34:56",
  "status": 400,
  "error": "Bad Request",
  "message": "잘못된 입력 값입니다."
}
----
